CEWAM

The Circular Economy Wind Agent-based Model (CEWAM) is an agent-based model built with the MESA python library. The tool enables studying various circular economy strategies by simulating actors decisions.

Features

    Based on the USWTDB
    Network configuration
    Product growth and waste generation
    Agents end-of-life decision rule

Requirements

    MESA
    networkx
    ...

Warnings

    The WBJ_Landfills_2020.csv file is proprietary, alternatively the code may be adpated to use the EPA 
    LMOP database which is publicly available
